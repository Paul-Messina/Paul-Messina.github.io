Installing React

We first started by installing homebrew with this command: /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" To install React, we used the homebrew command to install the npm package manager from the terminal: `brew install npm` From there, to get our project started, we simply typed `npx create-react-app TopTapLandingPage` and the boilerplate files were created in our current directory.

Installing Node.js

Installing node js was a very simple process as well. Using the npm command in the terminal: `npm install node` allowed us to run node.js programs on our local machines. (As well, we could have used the command `brew install node` using homebrew as the package manager.)
